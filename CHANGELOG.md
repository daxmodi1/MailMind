## [0.1.0] - 2025-12-06

### Added
- AI-powered email summarization using Groq LLM (llama-3.3-70b-versatile)
- Gmail integration with full read/write access
- Smart search with real-time highlighting
- Email statistics dashboard with daily activity charts
- Compose, reply, and forward emails
- Confidential mode for sensitive emails
- Draft auto-save functionality
- "Help Me Write" AI assistant for composing emails
- Responsive sidebar navigation
- Dark mode support
- Coming Soon page for Pro features
- Professional documentation (README, CONTRIBUTING, etc.)

### Security
- OAuth 2.0 authentication via Google
- Session management with NextAuth.js
- Server-side API routes for sensitive operations
